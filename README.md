# 끼니끼리

> 원하는 사람끼리 소중한 한 끼니



## 개요

공통 관심사를 기반으로 같이 활동할 수 있는 모임을 조성해주는 사이트로, 직업적 역량 향상 및 취미 개발 등 자기 성장을 원하는 사람들이 직접 만나 활동을 체험하거나 스터디한다.(보완하기)
